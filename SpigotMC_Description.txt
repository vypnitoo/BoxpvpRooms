## üìã **COMMANDS**

[CODE]
Main Commands:
/rooms wand [clan/team]        - Get creation tool
/1v1rooms create <name>        - Create new arena
/bpr edit <name>               - Edit existing arena
/rooms delete <name>           - Delete arena
/rooms mode <zone/barrier/teleport> - Change selection mode
/rooms save                    - Save current arena
/rooms cancel                  - Cancel current edit
/rooms reload                  - Reload configuration

Property Management:
/rooms prop <room> keepinventory <true/false>  - Keep inventory setting
/rooms prop <room> combatlog <true/false>      - Combat log protection
/rooms prop <room> preventblocks <true/false>  - Block protection
/rooms prop <room> barrier <material>          - Barrier material
/rooms prop <room> effect add <effect> <level> - Add potion effect
/rooms prop <room> effect remove <effect>      - Remove effect
/rooms prop <room> effect clear                - Clear all effects

Statistics:
/rooms stats [player]          - View player statistics
/rooms leaderboard [category]  - View leaderboards

Aliases: /1v1rooms, /bpr work for all commands!
[/CODE]
